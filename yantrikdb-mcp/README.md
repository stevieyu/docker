# [yantrikdb-mcp](https://github.com/yantrikos/yantrikdb-mcp)

当今的人工智能系统没有统一的内存架构。它们只是将各种通用数据库——向量存储、知识图谱、键值缓存——拼凑在一起，而这些数据库都不是为认知的工作原理而设计的。

人工智能需要一个专用的内存引擎，并原生支持以下功能：

- 时间衰减 ——记忆会像人类记忆一样老化和消退
- 语义整合 ——提取模式，压缩冗余
- 冲突解决 ——通过对话发现并解决矛盾。
- 主动认知 ——后台处理，它为人工智能提供启动的真正理由
- 多设备复制 ——基于本地优先 CRDT 的同步
所有功能都集成在一个嵌入式引擎中——无需服务器，无需网络跳转。


## Runner

```bash
docker build -t yantrikdb-mcp .

docker run -p 8420:8420 yantrikdb-mcp

# http://127.0.01:8420/mcp
```
